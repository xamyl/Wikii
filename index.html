Wikii Docs

This tool allows you to easily generate a wiki website from markdown files. It comes with built-in search and a simple server to serve your generated wiki.

Features

  • Convert markdown files to HTML.
  • Built-in search with Lunr.js.
  • Simple Express.js server to serve your wiki.

This documentation will guide you through using Wikii.
